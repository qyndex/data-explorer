# Data Explorer

Upload CSV/JSON files, explore data with tables, visualize with charts, and export results.

**Stack:** Fastapi | **Complexity:** moderate
**Tags:** `data`, `explorer`, `csv`, `json`, `visualization`, `pandas`, `plotly`

## Quick Start

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Docker

```bash
docker compose up --build
```
